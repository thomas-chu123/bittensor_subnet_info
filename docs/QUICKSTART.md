### 📦 第 1 步：安裝依賴

```bash
cd /Users/skynet/PycharmProjects/bittensor_subnet_info
```


## 🎮 使用場景

### 📊 場景 1：查看所有 Subnet 的投資價值

1. 頁面加載後自動顯示所有 Subnet
2. 查看「投資價值」列，綠色表示低風險，紅色表示高風險
3. 按「投資價值 (高→低)」排序查看最優機會

### 🔍 場景 2：過濾高質量投資機會

1. 調整頁面上方的「投資價值範圍」滑塊
2. 設置範圍為 **0.7 ~ 1.0** 查看低風險 Subnet
3. 查看「平均投資價值」和「顯示 Subnet 數」統計

### 💰 場景 3：按成本排序尋找便宜 Subnet

1. 點擊「排序方式」下拉菜單
2. 選擇「Registration Cost (低→高)」
3. 在頂部找到成本最低的 Subnet

### 📥 場景 4：導出數據進行分析

1. 調整過濾條件找到感興趣的 Subnet
2. 點擊「📥 導出」按鈕
3. 獲得 JSON 格式文件，可用 Excel/Python 進一步分析

## 📊 投資價值指標詳解

| 投資價值 | 風險等級 | 特徵 | 建議 |
|--------|--------|------|------|
| 🟢 0.8~1.0 | 低風險 | Incentive 分佈均勻 | ✅ **推薦投資** |
| 🟢 0.6~0.8 | 中低風險 | 大部分 validators 有獎勵 | ✅ **可考慮** |
| 🟡 0.4~0.6 | 中風險 | 少數頂級 validators 獲得大部分 | ⚠️ **謹慎** |
| 🟠 0.2~0.4 | 中高風險 | 集中度高 | ❌ **不推薦** |
| 🔴 0.0~0.2 | 高風險 | 極度集中或無數據 | ❌ **避免** |

**計算公式**: `投資價值 = 1.0 - Gini係數`

**Gini 係數**衡量 incentive 分佈的均勻程度：
- Gini = 0：完全均勻（安全）
- Gini = 1：完全集中（風險）

## ❓ 常見問題

**Q: 為什麼數據加載很慢？**
> A: 首次加載需要並行獲取 50+ Subnet 的區塊鏈數據。這是正常的（15~20秒）。使用「🔄 加載數據」按鈕可手動刷新。

**Q: 投資價值是什麼意思？**
> A: 基於 Incentive 分佈的均勻程度。高值 (0.8+) = 所有 validators 都能獲得獎勵；低值 (0.2-) = 只有少數頂級 validators 獲得獎勵。

**Q: 我可以導出數據嗎？**
> A: 是的！點擊「📥 導出」按鈕下載 JSON 格式數據。

**Q: Phase 2 的 SSH/RunPod 功能何時發布？**
> A: 開發中。目前可在儀表盤上看到按鈕佔位符。

**Q: 可以部署到遠端伺服器嗎？**
> A: 是的！使用：
> ```bash
> python3 -m uvicorn main:app --host 0.0.0.0 --port 8000 --workers 4
> ```
